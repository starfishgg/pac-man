# Pac-Man Clone (Pygame)

A simple Pac-Man game built with Python and Pygame.

## Features
- (Basic, to be improved) Ghost AI
- Sound effects
- Classic maze or Ms.Pacman maze loaded from file either:
- data/pacman.txt
- data/mspacman.txt
-  You can configure which map you want to load via the settings.py
- A few other things you can change via the settings such as:
- movement speed
- window size
- toggleable ghost names
  

## How to run
```bash
python main.py
